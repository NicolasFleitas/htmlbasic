Mi primer svg

Qué es SVG?

  1. SVG significa Scalable Vector Graphics
  2. SVG se usa para definiar un grafico basado en vectores para la web
  3. SVG define los gráficos en formato XML
  4. Cada elemento y cada atributo en los archivos SVG pueden ser animados
  5. Es una recomendación de W3C
  6. SVG se integra con otros estándares W3C como DOM y XSL

Ventajas de SVG

  1. Las imágenes SVG se pueden crear y editar con cualquier editor de texto
  2. Las imágenes SVG se puden buscar, indexar, crear secuencias de comando y comprimir
  3. Las imágenes SVG son escalables
  4. Las imágenes SVG se pueden imprimir con alta calidad en cualquier resolución
  5. Las imágenes SVG son con zoom
  6. Los gráficos SVG NO pierden ninguna calidad si se amplian o redimensionan
  7. SVG es un estándar abierto
  8. Los archivos SVG son XML puro

Su principal competidor es Flash

La myor ventaja que tiene SVG sobre Flash es el cumplimiento con otros estándares (por ejemplo, XSL y DOM) Flash se basa en tecnología patentada que no es de código abierto

Se pueden crear con cualquier editor de texto, pero es conveniente crearles con un programa de dibujo, como Inkscape.

SVG Path

El elemento path es usaro para definar una ruta

Los siguientes comandos estan disponibles para datos de ruta

  • M = moveto
  • L = lineto
  • H = horizontal lineto
  • V = vertical lineto
  • C = curveto
  • S = smooth curveto
  • Q = quadratic Bezier curve
  • T = smooth quadratic Bezier curveto
  • A = elliptical Arc
  • Z = closepath

NOTE: Todos los comandos mencionados tambien se pueden expresar en minusculas.
Letras Capitales significa absolutamento posicionado.
Minusculas significa relativamente posicionado
